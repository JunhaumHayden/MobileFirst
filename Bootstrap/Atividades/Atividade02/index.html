Desenvolvimento para Dispositivos Móveis

Se você tem interesse em aprender a criar aplicativos e sistemas para dispositivos móveis, este curso é para você!

No curso técnico de Desenvolvimento de Sistemas do IFSC - Campus Florianópolis, oferecemos disciplinas como Desenvolvimento para Dispositivos Móveis, onde você aprenderá a construir aplicações modernas utilizando tecnologias web, como HTML5, CSS3, e JavaScript.

O que você vai aprender:

  • Desenvolvimento Mobile First
  • Responsividade com CSS
  • Utilização de frameworks como Bootstrap
  • Desenvolvimento de Progressive Web Apps (PWA)

    • Venha e acenda a luz do conhecimento conosco.
